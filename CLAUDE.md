# ISIL — Site internet

Site vitrine pour ISIL, service professionnel de démoustication au Pays Basque / Sud Landes.
Technologie : HTML/CSS/JS vanilla, zéro framework.

## Stack
- Fonts : Cormorant Garamond + DM Sans + Barlow Condensed (Google Fonts)
- CSS custom properties : --cream #ECE7D8, --cream-l #F7F4ED, --navy #1A2B35, --green #7DB84A, --blue #3A7593
- Scroll reveal via IntersectionObserver (.rv / .rv.in)
- Splash page vidéo : accueil.html → redirige vers index.html

## Pages
- accueil.html — splash page vidéo (entrée du site)
- index.html — homepage avec hero plein écran
- probleme.html — page Le Moustique
- solution.html — page Notre Solution (flip 3D produit)
- offres.html — page Nos Offres
- contact.html — formulaire contact
