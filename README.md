# ISIL — Site web

Installation et maintenance de pièges Biogents anti-moustiques tigres au Pays Basque et dans le Sud des Landes.

## Structure

- `index.html` — page d'accueil (landing)
- `accueil.html`, `probleme.html`, `solution.html`, `offres.html`, `contact.html` — pages du site
- `flyer.html` — flyer imprimable
- `styles.css` — feuille de style
- `script.js` — interactions
- `images/` — ressources visuelles
- `isil-logo-trans.png`, `logo en action.mp4` — identité visuelle

## Déploiement

Site statique — tous les fichiers sont servis tels quels. Point d'entrée : `index.html`.
